# Dashcam Vehicle State Classification

A machine learning project to classify vehicle states from dashcam image frames using feature-based methods.

## Overview
This project aims to classify the state of a vehicle from dashcam frames into categories such as AHEAD, HALTED, SHARP_LEFT, and SHARP_RIGHT.

Instead of deep learning, the focus was on extracting meaningful visual features and using traditional machine learning models for classification.

## Objectives
- Extract useful features from image frames
- Train classification models on labeled data
- Evaluate model performance
- Analyze challenges like class imbalance

## Methods Used
- Grayscale conversion
- Edge detection
- Blur-based features
- Frame intensity analysis
- Feature engineering

## Tools and Libraries

- NumPy
- Pandas
- OpenCV
- Scikit-learn
- Matplotlib

## What I Learned
